Quando utilizar CSS Grid em vez de Flexbox:

Grid é mais adequado quando você está lidando com layouts bidimensionais, ou seja, layouts que envolvem tanto colunas quanto linhas. Se você precisa controlar o alinhamento e o tamanho de itens ao longo de ambas as direções (horizontal e vertical) de maneira simultânea, CSS Grid é a melhor escolha.
